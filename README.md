## 🌌 Overview

Welcome to my **Portfolio V2.0**. This application serves as my digital footprint, blending artificial intelligence elements with a premium, futuristic cyberpunk design aesthetic. The entire architecture relies on smooth micro-animations, a highly reactive UI, and a custom interactive cursor experience to deliver an absolute "wow" factor upon entry. 

**Demo Live On Vercel (Coming Soon!)**

---

## ✨ Features

- **🚀 Performance First**: Built with Vite and React 19 for instantaneous hot module reloading and incredibly fast production bundles.
- **🎨 Premium Cyberpunk Aesthetics**: A highly custom design system leveraging Tailwind CSS with custom neon glowing effects, glassmorphic (`backdrop-blur`) panels, and deep, dark ambient cyber colors.
- **💫 Interactive Animations**: Powered by Framer Motion, featuring buttery smooth page transitions, element staggering, hover animations, and a dynamic AI background orb.
- **⚡ Custom Cursor Glow**: A custom, real-time reactive cursor built from scratch that follows your interactions throughout the layout.
- **📱 Fully Responsive**: Fluid layouts that look flawless on ultra-wide desktop monitors all the way down to small mobile screens.
- **⌨️ Typewriter Effects**: Engaging text animations using `react-type-animation`.
- **🧩 Scroll-Spy Navigation**: The sidebar and navbar automatically highlight depending on the current active scrolling section.

## 💻 Running Locally

To run this futuristic experience on your local machine:

**1. Clone the repository:**
```bash
git clone https://github.com/ANASF1412/AnasF_Portfolio.git
cd AnasF_Portfolio
```

**2. Install dependencies:**
```bash
npm install
```

**3. Initialize the development server:**
```bash
npm run dev
```

**4. View in browser:**
The system is now online. Navigate to `http://localhost:5173/` in your browser.

---

## 📦 Building for Production

To create a highly optimized, minified production build:

```bash
npm run build
```
This generates your production-ready files in the `dist` folder. You can test them locally using `npm run preview`.
